# Portfolio Site (React + Vite + Cloudflare)

個人ポートフォリオサイトです。  
React + Vite で構築し、Cloudflare Workers (Wrangler) で配信しています。
https://portfolio.yuge-a-24kdgh.workers.dev/
リンクですぜひご覧ください

## Tech Stack

- React 18
- Vite 7
- Tailwind CSS 4
- React Router
- Framer Motion
- Cloudflare Wrangler


## Scripts

- `npm run dev`: Vite 開発サーバー
- `npm run build`: 本番ビルド
- `npm run preview`: Cloudflare ローカル確認
- `npm run deploy`: Cloudflare へデプロイ
- `npm run lint`: ESLint 実行
- `npm run prettier`: Prettier 整形
